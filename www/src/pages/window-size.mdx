# WindowSize

Calls back with the window dimensions.

```jsx
<WindowSize>
  {size => (
    <div>
      <div>Resize your window.</div>
      <pre>{JSON.stringify(size, null, 2)}</pre>
    </div>
  )}
</WindowSize>
```

## Installation

```bash
npm install @reach/window-size
# or
yarn add @reach/window-size
```

And then import it:

```js
import WindowSize from "@reach/window-size"
```

## Props

| Prop                  | Type | Required |
| --------------------- | ---- | -------- |
| [children](#children) | func | true     |

### children

_Type_: `func: (({ width, height })) => node`

A function that calls back to you with the window size.
